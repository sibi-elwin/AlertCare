## Project Summary
Silent Deterioration Detector (SDD) is a modern, clinical-grade yet human-centric web application designed to detect loss of health stability before health fails. It serves two primary roles: Patients (Elderly users) and Caregivers (Clinicians/Family). The system focuses on predictive intelligence, stability over time, and calm authority.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Components**: Radix UI, Shadcn UI patterns
- **Charts**: Recharts

## Architecture
- `src/app`: Page routes and layouts
- `src/components`: Reusable UI components
- `src/lib`: Utility functions and hooks

## User Preferences
- Dark-mode first aesthetic (deep slate / charcoal)
- Muted cyan → teal → soft amber accent gradients
- Glassmorphism panels with subtle blur
- High-legibility typography for elderly readability
- Numerical data emphasized with monospaced numerals

## Project Guidelines
- No sharp reds or flashing alerts
- Smooth physics-based animations
- One question per screen for patient onboarding
- Role-based navigation structure

## Common Patterns
- Framer Motion for scroll-triggered animations and transitions
- Glassmorphism for panel backgrounds
- Tailwind for responsive and accessible design
